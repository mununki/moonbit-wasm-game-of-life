## Deps

This example works with the following versions of the environment.

```
❯ moon version
moon 0.1.0 (bb920c0 2023-08-18)

❯ wat2wasm --version
1.0.33 (git~1.0.33-23-g3d1d29f6)
```

## Run

make run
